# Programmatic Audience Segmentation Tool

A Python application that uses NLP and unsupervised learning to segment advertising prospects from unstructured data, then outputs actionable audience profiles for sales teams.

**Built by CJ Fleming** — media sales leader with 15+ years of experience and a Columbia University AI certification.

---

## What This Demonstrates

| Skill Area | Implementation |
|---|---|
| **NLP** | TF-IDF vectorization of free-text company descriptions and campaign activity with bigram extraction |
| **Unsupervised Learning** | K-Means clustering with elbow method for optimal k, DBSCAN comparison |
| **Dimensionality Reduction** | UMAP for neighbourhood-preserving projection (PCA fallback) |
| **Data Pipeline** | End-to-end: raw CSV → feature engineering → clustering → profiling → outputs |
| **Production Python** | Type hints, docstrings, argparse CLI, modular package structure, cached Streamlit app |

## Quick Start

### 1. Install dependencies

```bash
cd audience-segmentation-tool
pip install -r requirements.txt
```

### 2. Generate synthetic data

```bash
python generate_data.py
```

This creates `data/prospects.csv` (500 records) and `data/deal_history.csv` (200 records).

### 3. Run the segmentation pipeline (CLI)

```bash
python segment.py --input data/prospects.csv --clusters auto --output results/
```

#### CLI Options

| Flag | Description | Default |
|---|---|---|
| `--input` | Path to prospects CSV | `data/prospects.csv` |
| `--clusters` | Number of clusters (`auto` or integer) | `auto` |
| `--output` | Output directory | `results/` |
| `--no-umap` | Use PCA instead of UMAP | UMAP enabled |
| `--no-dbscan` | Skip DBSCAN comparison | DBSCAN enabled |
| `--verbose` / `-v` | Debug logging | Off |

### 4. Explore results in Streamlit

```bash
streamlit run app.py
```

The Streamlit app provides:
- Interactive cluster scatter plot
- Segment profile cards with key metrics
- Filterable company explorer
- Outreach strategy recommendations per segment

- **segment_profiles.json** — Per-cluster statistics: median revenue, dominant industry, campaign themes, top demographics

- **report.md** — Markdown report with tables, industry breakdowns, and outreach recommendations

## Technical Details

- **TF-IDF**: Unigrams + bigrams, max 500 features per text column, English stop words removed
- **UMAP**: 10-component reduction with cosine metric for clustering; separate 2-component projection for visualization
- **K-Means**: Elbow method scans k=2..15, selects knee point; silhouette score logged for validation
- **DBSCAN**: Run with eps=2.0 and min_samples=5 as a density-based comparison
- **Segment Naming**: Rule-based system combining revenue tier, dominant industry, and engagement qualifiers
- **Outreach Strategies**: Template selection based on revenue × social-engagement quadrant
